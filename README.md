# Project:
```bash
Remember, if you download this project, please add everything into a new root folder and install every dependencies with ```pnpm i``` on all folders.

Relay ( Synk ) has a lot of different functions, so feel free to explore its code. You will find some cool stuff such as middlewares, ratelimit and throttling, http request, api requests, queries, auth... and so on!

This project is deployed on:

https://synk.xn--q9jyb4c/    (this one is for admin function and login page)
https://synk.xn--q9jyb4c/menu?tableId=1   (this route is for clients), to access other tables you need to change tableId=X. And please check if the table exists (you can check on:  https://synk.xn--q9jyb4c/console/dashboard).

Please consider leaving a ⭐and giving me a feedback. As this was my first "big" project that I made. Thanks!
```
